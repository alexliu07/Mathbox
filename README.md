# 数学工具箱<br>Mathbox

一个集成一些数学工具的工具箱

------

*我用c++写了一些数学有关的程序，但是每次想用的时候都要打开IDE再运行，非常麻烦，刚好最近接触了一点java，于是就想在手机上实现这些功能*

------

预计完成5个工具后发布release

## 功能

1. 根式化简<br>对任意次数的根式进行化简，而不是显示出近似小数
2. 寻找因数<br>寻找乘积为指定整数的所有整数
3. 寻找公因数<br>寻找两个数的公因数
4. 分数约分
5. 一元二次方程求解

------
